# 国际化

游戏支持中文（zh-CN）和英文（en-US）两种语言，使用 [i18n-embed](https://crates.io/crates/i18n-embed) 和 Fluent 翻译系统。

## 翻译文件结构

```
locales/
  en-US/
    main.ftl        # 英文翻译
  zh-CN/
    main.ftl        # 中文翻译
```

## Translator 封装

`core/i18n.rs` 提供了 `Translator` 类型，封装翻译查找：

```rust
let t = Translator::new(language);
t.text("button-new-game")                    // 简单文本
t.text_args("message-turn-finished", &args([("count", "5")]))  // 带参数
```

## 设计决策

| 决策 | 理由 |
|------|------|
| Fluent 而非 gettext | Rust 生态友好，支持参数化和复数 |
| rust-embed 内嵌 | 打包后不依赖外部 locale 文件 |
| 语言切换在运行时 | 通过设置面板即时切换，无需重启 |

## 翻译范围

- UI 标签、按钮、提示文字 — 翻译
- 游戏内历史名称（武将名、城池名）— 不翻译，保持原文
- 系统消息和报告 — 翻译
