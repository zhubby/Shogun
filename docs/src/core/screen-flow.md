# 屏幕流程

游戏有两个主屏幕，由 `Screen` 枚举控制：

```rust
pub enum Screen {
    MainMenu,
    InGame,
}
```

## 流程图

```
                    ┌──────────────┐
                    │   启动 Bevy   │
                    └──────┬───────┘
                           ▼
                    ┌──────────────┐
              ┌────▶│   主菜单      │◀──────────────┐
              │     │  MainMenu     │               │
              │     └──┬───┬───┬───┘               │
              │        │   │   │                    │
              │   新局  │   │   │ 读档              │ 退出到菜单
              │        │   │   │                    │
              │        ▼   │   ▼                    │
              │   选剧本  设置  选存档               │
              │   选势力  面板  面板                  │
              │        │       │                    │
              │        ▼       ▼                    │
              │     ┌──────────────┐               │
              └─────│   游戏内      │───────────────┘
                    │    InGame     │
                    └──────────────┘
```

## 主菜单

主菜单提供以下功能：
- **新游戏** — 从历史资料库选择剧本和势力
- **读档** — 列出存档槽位，选择加载
- **设置** — 显示/音频/语言三个标签页
- **退出** — 退出应用

## 游戏内

游戏内界面由多个 HUD 层叠加：

```
┌──────────────────────────────────────────┐
│  顶部状态栏（势力、年月、资源、结束回合）   │
├────────┬─────────────────────┬───────────┤
│ 城池   │                     │ 地图控制  │
│ 摘要   │     地图面板         │           │
│ 面板   │   (缩放/平移/标记)   │           │
│        │                     │           │
├────────┤                     ├───────────┤
│ 城池   │                     │ 底部命令栏 │
│ 列表   │                     │           │
└────────┴─────────────────────┴───────────┘
     + 浮动面板：报告、存档、科技、武将浏览器
```
