# 城池与设施

城池是游戏中最基本的地理和经济单位。每个城池属于一个势力，由太守管理，包含多种设施。

## 城池结构

城池分为两个层面：

- **CityProfile** — 静态资料（来自历史数据库）：位置、省份、郡、规模、战略评级、经济基础值
- **City** — 动态状态（在 GameState 中）：归属势力、资源、设施、人口、治安、太守

## 城池规模（CityScale）

| 规模 | 说明 |
|------|------|
| `County` | 县城 |
| `Commandery` | 郡治 |
| `RegionalCapital` | 州治 |
| `ImperialCapital` | 帝都 |

规模影响设施槽位数和经济上限。

## 设施系统

城池可建设 12 种设施，每种最高 5 级：

| 设施 | 类别 | 效果 |
|------|------|------|
| 农田 (Farmland) | 农业 | 粮食收入 |
| 水利 (Irrigation) | 农业 | 农业加成 |
| 市场 (Market) | 商业 | 金钱收入 |
| 商栈 (TradeDepot) | 商业 | 商业加成 |
| 工坊 (Workshop) | 工业 | 物资收入 |
| 采石场 (Quarry) | 工业 | 城防材料 |
| 兵营 (Barracks) | 军事 | 征兵容量 |
| 校场 (DrillGround) | 军事 | 训练效果 |
| 城墙 (Walls) | 防御 | 防御加成 |
| 府衙 (Administration) | 内政 | 治安加成 |
| 粮仓 (Granary) | 后勤 | 粮食储备 |
| 驿站 (RelayStation) | 后勤 | 行军速度 |

设施槽位数由城池规模决定。

## 经济效果

城池的经济产出受多重因素影响：

- 基础值（来自 CityProfile）
- 城池核心等级（最高 10 级）
- 设施加成
- 科技加成（`TechnologyBonuses`）
- 官职加成（太守和朝官的效果）
- 武将能力值
